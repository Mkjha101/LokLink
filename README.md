# LokLink

LokLink is a hyperlocal community platform built using the MERN stack.

Current Status:
✅ Frontend MVP Complete
⏳ Backend Under Development

## Features

- Community Feed
- Lost & Found
- Marketplace
- Community Updates
- Events
- Messenger
- Clubs
- Emergency Contacts
- Navigation
- Admin Dashboard

## Tech Stack

Frontend:
- React
- TypeScript
- TanStack Router
- Tailwind CSS

Backend (Planned):
- Node.js
- Express
- MongoDB
- JWT
- Socket.IO
